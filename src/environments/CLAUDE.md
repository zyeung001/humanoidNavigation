# src/environments/ — Gym Environments + Curriculum

All environments wrap `gymnasium.make("Humanoid-v5")` with custom observation processing, reward shaping, and curriculum advancement.

## Observation Space

| Component | Dims | Notes |
|-----------|------|-------|
| Humanoid-v5 base | 365 | Joint positions, velocities, forces (350 raw + 15 extra) |
| COM features | +6 | Center of mass position (3) + velocity (3) |
| Velocity command | +2 or +9 | Walking only: 2 appended per-frame, 9 in command block |
| History stacking | ×4 | Temporal context for velocity estimation |
| **Standing total** | **1484** | (365 + 6) × 4 |
| **Walking total** | **1493** | 1484 + 9 command block |

Command block (9 dims, appended once, not per-frame):
`[vx_cmd, vy_cmd, yaw_cmd, vx_actual, vy_actual, err_vx, err_vy, err_speed, err_angle]`

## Action Space

17 continuous actions (joint torques), range [-1, 1]. Action smoothing via EMA (`tau` configurable).

## Files

### `standing_env.py` — StandingEnv

Base standing (balance) environment.

**Reward components:**
1. Height reward: Asymmetric piecewise, peak at 1.40m (max 50)
2. Upright: `12 × exp(-8 × error²)` if height ≥ 1.2m
3. Stability: `exp(-2 × angular_momentum)` scaled by height
4. Smoothness: `2.5 × exp(-0.1 × ||joint_vel||²)`
5. Control cost: `-0.005 × ||action||²`
6. Height maintenance: `-150 × velocity` (falling penalty, capped at -15)
7. Recovery bonus: `+50 × velocity × scale` (rising from low height)
8. Sustained bonus: +100 every 100 steps if height_error < 0.08 and height ≥ 1.32m
9. Velocity penalty: `-1.0 × clip(speed - 0.5, 0, 2.0)` (drift penalty)

**Termination:**
- height < 0.75m (fell)
- height > 2.0m (launched)
- |q_w| < 0.6 (tipped over)

**Domain randomization:** mass ±5%, friction ±5% (configurable).
**Random height init:** 30% probability, range [-0.3, 0.1] from target.

### `standing_curriculum.py` — StandingCurriculumEnv(StandingEnv)

6-stage progressive height curriculum:

| Stage | Target | Tolerance | Min Episode Len | Domain Rand |
|-------|--------|-----------|-----------------|-------------|
| 0 | 0.80m | ±0.25m | 80 | None |
| 1 | 1.00m | ±0.20m | 100 | None |
| 2 | 1.15m | ±0.15m | 200 | None |
| 3 | 1.25m | ±0.12m | 400 | Light (±2%) |
| 4 | 1.35m | ±0.10m | 800 | Light (±2%) |
| 5 | 1.40m | ±0.08m | 1200 | Full (±5%) |

**Advancement:** Height within tolerance + episode length ≥ minimum + not fallen + 60% success rate over 20 episodes.

### `walking_env.py` — WalkingEnv(gym.Wrapper)

Command-conditioned walking with velocity tracking.

**Reward:** Uses `RewardCalculator` from `src.core.rewards`. Tracking weight (25.0) is dominant signal.

**Command input:** Generated by `VelocityCommandGenerator`. Pre-normalized to [-1, 1] for policy.

**Perturbations:** Push forces (10-100 N), random height init (15%), random velocity init (±0.3 m/s).

### `walking_curriculum.py` — WalkingCurriculumEnv(WalkingEnv)

3-stage speed curriculum:

| Stage | Max Speed | Dir Diversity | Perturbations | Domain Rand |
|-------|-----------|---------------|---------------|-------------|
| 0 | 0.15 m/s | Forward only | None | None |
| 1 | 0.40 m/s | Limited | Light (10-40 N) | Light (±3%) |
| 2 | 0.80 m/s | Full | Heavy (20-80 N) | Full |

**Advancement (multiple paths):**
- Standard: success_rate ≥ 35% AND avg_vel_error < tolerance × 1.3
- Length-based: avg_length > 3× minimum AND fall_rate < 10%

**Anti-exploit (Stage 0):** Requires `actual_speed ≥ 0.5 × commanded_speed`. Prevents standing-still from passing.
